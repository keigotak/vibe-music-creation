# 不要トラックを削除

Ableton Liveプロジェクトからクリップが1つもない空のトラックを削除する。

## 手順

1. `get_project_table` で現在のプロジェクト構成を取得する
2. 全シーンにクリップが存在しないトラック（全て `-` のカラム）を特定する
3. 該当トラックを **インデックスの大きい順** に `osc_send` で削除する
   - アドレス: `/live/song/delete_track`
   - 引数: `[トラック番号]`（0始まり）
   - 大きい順に削除しないとインデックスがずれるので注意
4. 削除後、再度 `get_project_table` で結果を確認して表示する
5. 削除したトラック数を報告する。空トラックがなかった場合は「不要トラックはありませんでした」と報告する
