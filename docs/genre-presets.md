# ジャンル別プリセット/テンプレート

各ジャンルのトラック構成、音源、エフェクトチェーン、BPM、コード進行をまとめた一発立ち上げ用リファレンス。

---

## Lo-Fi Hip Hop

| 項目 | 設定 |
|------|------|
| BPM | 70-90（推奨78） |
| キー | Dm, Am, Cm が多い |
| 拍子 | 4/4 |

### トラック構成
| # | トラック | 音源 | エフェクトチェーン |
|---|---------|------|-------------------|
| 0 | Drums | Kit-Lo-Fi Tech | EQ Eight → Compressor → Saturator → Redux |
| 1 | Bass | Shy LoFi Keys | EQ Eight → Saturator → Compressor |
| 2 | Chords | E-Piano Wurli | Auto Filter → Chorus-Ensemble → Reverb → Vinyl Distortion |
| 3 | Melody | Piano系 / Basic Waves Lead | Auto Filter → Delay → Reverb → Erosion |
| 4 | Texture | (なし) | Vinyl Distortion (Crackle中心) |

### ミックス設定
| トラック | Volume | サイドチェイン |
|---------|--------|-------------|
| Drums | 0.85 | - |
| Bass | 0.75 | - |
| Chords | 0.65 | - |
| Melody | 0.55 | - |
| Texture | 0.3 | - |

### コード進行
- `Dm9 → G13 → Cmaj9` (ii-V-I)
- `Cmaj7 → Am7 → Dm7 → G7` (I-vi-ii-V)
- `Fmaj7 → Em7 → Am7 → Dm7` (IV-iii-vi-ii)
- `Fmaj7 → E7 → Am7` (IV-III7-vi)

### シーン構成
| # | シーン | 小節 | 内容 |
|---|--------|------|------|
| 0 | Intro | 4-8 | コード+テクスチャのみ、フィルター暗め |
| 1 | Main A | 8 | メインループ全要素 |
| 2 | Main B | 8 | メロディ変形、ドラム微変形 |
| 3 | Bridge | 4-8 | ドラム抜き、空間エフェクト増 |
| 4 | Main A' | 8 | メイン回帰+微変化 |
| 5 | Outro | 4-8 | フィルターダウン、フェードアウト |

## J-core (Japanese Hardcore)

| 項目 | 設定 |
|------|------|
| BPM | 170-185（推奨175） |
| キー | Am, Em が多い |
| 拍子 | 4/4 |

### トラック構成
| # | トラック | 音源 | エフェクトチェーン |
|---|---------|------|-------------------|
| 0 | Drums | 909 Core Kit | EQ Three → Glue Compressor |
| 1 | Sub Bass | Basic Sub Sine | Compressor, EQ（低域集中） |
| 2 | Donk Bass | 808 | Overdrive, EQ（中低域） |
| 3 | Chords Core | Mega Saw | EQ（300-500Hzカット） |
| 4 | Chords Wide | Celestial Strings | Reverb, EQ（300-500Hzカット） |
| 5 | Lead | Square Lead | EQ（800Hz付近カット）, Delay |
| 6 | Lead Sub | Superstring Lead | EQ（低中域に集中） |
| 7 | FX | Hyper Riser | - |

### ミックス設定
| トラック | Volume | サイドチェイン |
|---------|--------|-------------|
| Drums | 0.9 | - |
| Sub Bass | 0.8 | 0.6 |
| Donk Bass | 0.7 | 0.6 |
| Chords Core | 0.65 | 0.5 |
| Chords Wide | 0.55 | 0.5 |
| Lead | 0.7 | - |
| Lead Sub | 0.55 | - |
| FX | 0.5 | - |

### コード進行
- `Am → F → C → G` (vi-IV-I-V: 王道エモ)
- `Am → Em → F → C → G` (vi-iii-IV-I-V)
- `Fmaj7 → G → Am → Em` (IV7-V-vi-iii)

### シーン構成
| # | シーン | 小節 | 内容 |
|---|--------|------|------|
| 0 | Drop | 8 | フルエネルギー、キック刻み |
| 1 | Intro | 8 | HHのみ or キック控えめ |
| 2 | Buildup | 8 | スネアロール、Riser、フィルタースイープ |
| 3 | Breakdown | 4-8 | ドラム抜き、エモ最大化 |
| 4 | Buildup2 | 8 | Buildup変形 |
| 5 | Drop2 | 8 | メロディ変形Drop |
| 6 | Outro | 8 | 要素減少、Downlifter |

## Boom Bap / 90s Hip Hop

| 項目 | 設定 |
|------|------|
| BPM | 85-98（推奨92） |
| キー | Dm, Am, Gm が多い |
| 拍子 | 4/4 |

### トラック構成
| # | トラック | 音源 | エフェクトチェーン |
|---|---------|------|-------------------|
| 0 | Drums | SP-1200系 or 808 | EQ Eight → Compressor |
| 1 | Bass | ファンキーベース系 | EQ Eight → Saturator |
| 2 | Keys | エレピ/Rhodes系 | Auto Filter → Reverb → Vinyl Distortion |
| 3 | Stab/Sample | ブラスorストリングス | Delay → Reverb → Bitcrusher |
| 4 | Texture | (なし) | Vinyl Distortion |

### コード進行
- `Dm9 → Am7 → Bbmaj7 → A7` (i-v-bVI-V7: ジャジー)
- `Cm7 → Fm7 → Gm7 → Cm7` (i-iv-v-i: マイナーループ)
- `Fmaj7 → E7 → Am7 → Dm7` (IV-III7-vi-ii)

## Ambient / Chillout

| 項目 | 設定 |
|------|------|
| BPM | 60-80（推奨65） |
| キー | C, Am, F が多い |
| 拍子 | 4/4 or フリー |

### トラック構成
| # | トラック | 音源 | エフェクトチェーン |
|---|---------|------|-------------------|
| 0 | Pad | Celestial Strings / Ambient系 | Reverb(深い) → Delay |
| 1 | Texture | Granular系 | Reverb → Auto Filter |
| 2 | Sub Drone | Sine/Triangle | LPF → Reverb |
| 3 | Melody | ベル系 / Piano | Delay → Reverb(深い) |

### コード進行
- `Cmaj7 → Am7` (I-vi: 2コードループ)
- `Fmaj9 → Cmaj9 → Am9 → Em7` (IV-I-vi-iii: 浮遊系)
- 1コードにつき2-4小節（ハーモニックリズムを遅く）

## Drum and Bass

| 項目 | 設定 |
|------|------|
| BPM | 170-180（推奨174） |
| キー | Cm, Fm, Am が多い |
| 拍子 | 4/4 |

### トラック構成
| # | トラック | 音源 | エフェクトチェーン |
|---|---------|------|-------------------|
| 0 | Drums | 909 / Breakbeat系 | EQ Three → Glue Compressor |
| 1 | Reese Bass | Sawtooth系 | Auto Filter(LFO) → Compressor |
| 2 | Pad | Strings/Pad系 | Reverb → Delay |
| 3 | Lead | Supersaw / Neuro系 | Distortion → Auto Filter |

### コード進行
- `Cm → Fm → Gm → Cm` (i-iv-v-i)
- `Am → F → C → G` (vi-IV-I-V)

---

## 使い方
1. ジャンルを選ぶ
2. テンプレートに従ってOSC直接でトラック作成
3. コード進行を選択
4. シーン構成に沿ってクリップ作成
5. ミックス設定を適用
6. オートメーションで仕上げ
