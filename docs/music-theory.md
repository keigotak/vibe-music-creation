# 音楽理論・作曲リファレンス

SoundQuest (https://soundquest.jp/) を中心に体系化した音楽理論まとめ。
Lo-Fi Hip Hop制作に特化した実践的ガイド。

---

## 1. 音の基礎

### 音名と音程
- **12音**: C, C#/Db, D, D#/Eb, E, F, F#/Gb, G, G#/Ab, A, A#/Bb, B
- **半音(semitone)**: 隣り合う音の最小距離
- **全音(whole tone)**: 半音2つ分
- **オクターブ**: 12半音 = 同じ音名の1周期

### 音程(Interval)の名称
| 半音数 | 名称 | 略記 | 響き |
|--------|------|------|------|
| 0 | 完全1度(ユニゾン) | P1 | 同音 |
| 1 | 短2度 | m2 | 緊張 |
| 2 | 長2度 | M2 | やや緊張 |
| 3 | 短3度 | m3 | 暗い |
| 4 | 長3度 | M3 | 明るい |
| 5 | 完全4度 | P4 | 安定 |
| 6 | 増4度/減5度(トライトーン) | A4/d5 | 不安定 |
| 7 | 完全5度 | P5 | 最も安定 |
| 8 | 短6度 | m6 | 暗め |
| 9 | 長6度 | M6 | 明るめ |
| 10 | 短7度 | m7 | jazzy |
| 11 | 長7度 | M7 | おしゃれ |
| 12 | 完全8度(オクターブ) | P8 | 同音 |

## 2. スケール（音階）

### メジャースケール（長音階）
- 構造: 全全半全全全半 (W-W-H-W-W-W-H)
- Cメジャー: C D E F G A B
- 明るく安定した響き

### ナチュラルマイナースケール（自然的短音階）
- 構造: 全半全全半全全 (W-H-W-W-H-W-W)
- Cマイナー: C D Eb F G Ab Bb
- 暗く哀愁のある響き

### ハーモニックマイナースケール（和声的短音階）
- ナチュラルマイナーの第7音を半音上げたもの
- Cハーモニックマイナー: C D Eb F G Ab B
- 導音(B)→主音(C)の強い解決感を生む

### メロディックマイナースケール（旋律的短音階）
- 上行: ナチュラルマイナーの6度と7度を半音上げ
- 下行: ナチュラルマイナーに戻す（クラシック理論）
- ジャズでは上行形のみ使用

### ペンタトニックスケール
- **メジャーペンタ**: 1 2 3 5 6（4度と7度を抜く）
- **マイナーペンタ**: 1 b3 4 5 b7（2度と6度を抜く）
- Lo-Fiメロディで頻出。シンプルで外れにくい

### ブルーススケール
- マイナーペンタ + b5（ブルーノート）
- C: C Eb F Gb G Bb
- Lo-Fi的な哀愁とグルーヴ感

## 3. 教会旋法（チャーチモード）

SoundQuestでは7つの教会旋法を詳しく解説。
各モードはメジャースケールの各音を起点にしたスケール。

| モード | 起点 | 特性音 | 響き | 使用例 |
|--------|------|--------|------|--------|
| イオニアン | 1度 | なし | 明るい（=メジャー） | ポップス全般 |
| ドリアン | 2度 | ナチュラル6th | マイナーだが明るめ | Lo-Fi, ジャズ, ファンク |
| フリジアン | 3度 | b2nd | エキゾチック、暗い | フラメンコ, K-Pop |
| リディアン | 4度 | #4th | 浮遊感、幻想的 | 映画音楽, プログレ |
| ミクソリディアン | 5度 | b7th | ブルージー、明るい | ロック, ブルース |
| エオリアン | 6度 | なし | 暗い（=ナチュラルマイナー） | バラード, Lo-Fi |
| ロクリアン | 7度 | b2nd, b5th | 非常に不安定 | ほぼ使用されない |

**Lo-Fi Hip Hopで特に重要**: ドリアン、ミクソリディアン、エオリアン

## 4. コード（和音）の基礎

### トライアド（三和音）
| 種類 | 構成 | 例(C) | 響き |
|------|------|-------|------|
| メジャー | 1-3-5 | C-E-G | 明るい |
| マイナー | 1-b3-5 | C-Eb-G | 暗い |
| ディミニッシュ | 1-b3-b5 | C-Eb-Gb | 不安定 |
| オーグメント | 1-3-#5 | C-E-G# | 浮遊感 |

### セブンスコード（四和音）★Lo-Fi必須
| 種類 | 構成 | 例(C) | 表記 | 用途 |
|------|------|-------|------|------|
| メジャーセブンス | 1-3-5-7 | C-E-G-B | Cmaj7 | トニック、Lo-Fiの主役 |
| マイナーセブンス | 1-b3-5-b7 | C-Eb-G-Bb | Cm7 | サブドミ、Lo-Fiの要 |
| ドミナントセブンス | 1-3-5-b7 | C-E-G-Bb | C7 | ドミナント |
| ハーフディミニッシュ | 1-b3-b5-b7 | C-Eb-Gb-Bb | Cm7(b5) | ii of minor |
| ディミニッシュセブンス | 1-b3-b5-bb7 | C-Eb-Gb-A | Cdim7 | 経過和音 |

### テンションコード ★Lo-Fiの味付け
テンション = コードトーンの上にさらに3度堆積で重ねた音（9th, 11th, 13th）

| テンション | 音程 | 効果 |
|-----------|------|------|
| 9th | 2度の1オクターブ上 | おしゃれ、浮遊感 |
| 11th | 4度の1オクターブ上 | 広がり |
| 13th | 6度の1オクターブ上 | 豊かさ |

**アヴォイドノート**: テンションとして使うと不協和が強すぎる音
- メジャーコード上の11th（4度）→ 3度と半音でぶつかる
- ドミナントコード上の長3度と半音でぶつかる音

**Lo-Fiでよく使うテンション付きコード**:
- `Cmaj9` = C-E-G-B-D（メジャーセブンスに9thを追加）
- `Dm9` = D-F-A-C-E（マイナーセブンスに9thを追加）
- `Cadd9` = C-E-G-D（セブンスなしで9thだけ追加）

## 5. ダイアトニックコード（基調和音）

### Cメジャーキーのダイアトニックコード

SoundQuestでは「六つの基調和音（Prime Chords）」として解説。
vii°は不安定すぎるため実用上は6つが基本。

| 度数 | トライアド | セブンス | 機能 |
|------|-----------|---------|------|
| I | C | Cmaj7 | **トニック (T)** |
| ii | Dm | Dm7 | **サブドミナント (SD)** |
| iii | Em | Em7 | トニック代理 (T) |
| IV | F | Fmaj7 | **サブドミナント (SD)** |
| V | G | G7 | **ドミナント (D)** |
| vi | Am | Am7 | トニック代理 (T) |
| vii° | Bdim | Bm7(b5) | ドミナント代理 (D) |

### 機能和声（ファンクショナルハーモニー）

SoundQuestの「コードの機能」ページの核心概念。

**3つの機能**:
- **トニック (T)**: 安定・解決感 → I, iii, vi
- **サブドミナント (SD)**: やや不安定・移行感 → ii, IV
- **ドミナント (D)**: 最も不安定・トニックへ解決したい → V, vii°

**機能の流れ（基本法則）**:
```
T → SD → D → T （順行 = 自然な流れ）
T → D → T     （SD省略も可）
T → SD → T    （D省略でソフトな終止）
```

**逆行（D → SD）は基本的に避ける**（SoundQuestで強調）

### 代理コード

SoundQuestの「スリーコードと代理コード」より。
同じ機能を持つコード同士は代理（置き換え）可能。

- I ↔ iii, vi （トニック代理）
- IV ↔ ii （サブドミナント代理）
- V ↔ vii° （ドミナント代理）

**Lo-Fiでの活用**:
- `C → G → Am → F` の C を Am7 に代理 → `Am7 → G → Am → F` でマイナー感強化

---

## 6. ケーデンス（終止形）

SoundQuestの「ケーデンス(終止形)」より。
コード進行の「句読点」にあたるもの。

| ケーデンス | 進行 | 効果 |
|-----------|------|------|
| **正格終止 (Authentic)** | V → I | 最も強い解決感 |
| **変格終止 (Plagal)** | IV → I | 穏やかな解決（アーメン終止） |
| **半終止 (Half)** | ? → V | 一時停止感 |
| **偽終止 (Deceptive)** | V → vi | 意外性のある展開 |

**Lo-Fiでのポイント**:
- 正格終止は強すぎることが多い → **偽終止(V→vi)** や **変格終止(IV→I)** がLo-Fiの柔らかさに合う
- ループ進行では明確な終止を避け、循環感を出す

---

## 7. ノンダイアトニックコード

### セカンダリードミナント（二次ドミナント）

SoundQuestの「二次ドミナント」より。
ダイアトニックコードの各コードに対する「仮のV7」。

| 表記 | 解決先 | 例(Cメジャー) |
|------|--------|---------------|
| V7/ii | → ii (Dm) | A7 → Dm |
| V7/iii | → iii (Em) | B7 → Em |
| V7/IV | → IV (F) | C7 → F |
| V7/V | → V (G) | D7 → G |
| V7/vi | → vi (Am) | E7 → Am |

**ドミナントモーション**: V7の持つトライトーン（3度と7度の増4度音程）が半音で解決する力

### サブドミナントマイナー（SDm）

SoundQuestの「パラレル・マイナー」より。
同主調マイナー（CメジャーならCマイナー）から借用したコード。

**代表的なSDmコード**:
- **IVm** (Fm): 最も頻出。独特の哀愁、バラードの定番
- **bVI** (Ab): ドラマチックな響き
- **bVII** (Bb): ロック的な力強さ
- **IIm7(b5)** (Dm7b5): ジャジーな暗さ

**Lo-Fiでの活用**:
- `Cmaj7 → Fm7` のように IVm7 を使うと一瞬の切なさが生まれる
- 借用和音全般がLo-Fiの「ノスタルジー感」に大きく貢献

### モーダルインターチェンジ

パラレルキー（同主調）や他のモードからコードを借用する手法。
SoundQuestでは「パラレル・マイナー」「パラレル・メジャー」として解説。

**Cメジャーで使えるモーダルインターチェンジ**:
- Cドリアン由来: Bb, Gm
- Cミクソリディアン由来: Bb, Gm
- Cエオリアン由来: Ab, Bb, Eb, Fm
- Cフリジアン由来: Db, Bbm

## 8. コード進行パターン

### 定番進行（SoundQuestの接続系理論より）

**王道進行 (IV-V-iii-vi / 4536)**
```
F → G → Em → Am  (Cメジャー)
```
J-Pop/アニメで最頻出。SoundQuestでは「日本の王道進行」として解説。

**カノン進行 (I-V-vi-iii-IV-I-IV-V)**
```
C → G → Am → Em → F → C → F → G
```
パッヘルベルのカノン由来。ポップスの大定番。

**小室進行 (vi-IV-V-I / 6451)**
```
Am → F → G → C
```
マイナースタートで哀愁＋力強さ。

**Just the Two of Us進行 (IV-III7-vi / 4361)**
```
Fmaj7 → E7 → Am7 (→ Gm7 → C7)
```
セカンダリードミナント(E7→Am)を使ったジャジーな進行。

### Lo-Fi Hip Hop向けコード進行 ★実践

**基本: ii-V-I (ツーファイブワン)** ジャズの最重要進行
```
Dm9 → G13 → Cmaj9
```
Lo-Fiではセブンス/ナインスで装飾するのが定番。

**パターン1: シンプル Jazzy Loop**
```
Cmaj7 → Am7 → Dm7 → G7
(I → vi → ii → V)
```

**パターン2: マイナー系 Lo-Fi**
```
Am7 → Dm7 → Gmaj7 → Cmaj7
(vi → ii → V → I) ※マイナー始まり
```

**パターン3: 借用和音入り**
```
Fmaj7 → Em7 → Am7 → Abmaj7
(IV → iii → vi → bVI) ※bVIが借用和音
```

**パターン4: SDm活用**
```
Cmaj7 → Fm7 → Cmaj7 → G7
(I → IVm → I → V) ※Fm7がサブドミナントマイナー
```

**パターン5: ドリアン系**
```
Dm7 → G7 → Dm7 → G7
(i → IV) ※Dドリアンモード
```

**パターン6: 9th中心の浮遊系**
```
Fmaj9 → Em7 → Dm9 → Cmaj9
(IV → iii → ii → I) ※下降進行
```

**パターン7: Just the Two of Us風**
```
Dbmaj7 → C7 → Fm7 → Ebm7 → Ab7
```

### コード進行設計の原則（SoundQuestの接続系理論）

1. **5度下行**: 最も自然で強い進行（V→I, ii→V, vi→ii）
2. **2度上行/下行**: なめらかな進行
3. **3度下行**: トニック→代理の関係（I→vi, IV→ii）
4. **3度上行**: やや意外性のある動き

## 9. 転調（モジュレーション）

SoundQuestの「ポップスの転調技法」シリーズより。

### 転調の種類
| 転調 | 方法 | 効果 |
|------|------|------|
| **半音上** | Cメジャー→Dbメジャー | 高揚感（サビ前など） |
| **全音上** | Cメジャー→Dメジャー | 大きな高揚感 |
| **短3度下** | Cメジャー→Aメジャー | 自然な変化 |
| **短3度上** | Cメジャー→Ebメジャー | ダーク、ドラマチック |
| **同主調** | Cメジャー↔Cマイナー | 明暗の切り替え |
| **平行調** | Cメジャー↔Aマイナー | 最も自然 |

### ピボットコード
SoundQuestの解説: 転調前後の両方のキーに属するコードを「ピボット（軸）」として使い、自然に転調する手法。

例: CメジャーからGメジャーへ
- Em (CメジャーのiiiかつGメジャーのvi) をピボットに使用

## 10. ボイシングとボイスリーディング

### ボイシング（和音の配置）
SoundQuestの「ジャズのヴォイシング基礎」より。

- **クローズドボイシング**: 1オクターブ以内に全音を収める
- **オープンボイシング**: 音を広く分散させる（Lo-Fiピアノに最適）
- **ドロップ2**: クローズドの上から2番目の音を1オクターブ下げる

**Lo-Fiのボイシング Tips**:
- ルートは左手、3rd+7th(+テンション)を右手
- 5thは省略してよい（個性に乏しいため）
- 密集よりも開離の方がLo-Fi的な「ゆるさ」が出る

### ボイスリーディング（声部の進行）
- 共通音はなるべく保持する
- 各声部の動きは最小限にする（跳躍を避ける）
- 隣接するコード間で半音または全音の動きを意識

## 11. リズム理論

### 基本
- **拍子**: 4/4拍子が最も一般的（Lo-Fiもほぼ全て4/4）
- **テンポ**: Lo-Fiは通常 70-90 BPM
- **ハーモニックリズム**: コードが変わるペース
  - Lo-Fi: 2拍〜2小節に1コードが一般的

### Lo-Fi Hip Hopのリズム特徴
- **スウィング/シャッフル**: ハイハットに微妙な揺れ（50-60%のスウィング）
- **レイドバック**: スネアを少し遅らせる（グルーヴ感）
- **サイドチェーン**: キックに合わせてコンプレッション（うねり効果）
- **テープ劣化**: ピッチの微妙な揺れ（ワウ/フラッター）

## 12. メロディ理論

SoundQuestの「メロディ理論をはじめる」より。

### メロディの基本原則
- **順次進行**: 隣の音に進む（なめらか）
- **跳躍進行**: 3度以上飛ぶ（インパクト）
- **跳躍後は反対方向に順次進行**: 自然な流れ

### コードトーンとノンコードトーン
- **コードトーン**: コード構成音に着地 = 安定
- **パッシングトーン**: コードトーン間を経過する音
- **ネイバートーン**: コードトーンの隣の音から戻る
- **アプローチノート**: ターゲット音の半音/全音下から上行

### Lo-Fiメロディの特徴
- マイナーペンタ/ブルーススケールベース
- 音数は少なめ（スペースが重要）
- リピート/ループ感のあるフレーズ
- テンションを含むコードトーンへの着地（9thなど）

## 13. Lo-Fi Hip Hop 制作チェックリスト

### コード
- [ ] セブンスコード（maj7, m7）を基本にする
- [ ] 9thテンションで「おしゃれ」感を加える
- [ ] 4〜8小節のループ進行
- [ ] 借用和音(bVI, IVm)で切なさを演出
- [ ] ボイシングはオープン系（音を広げる）

### リズム
- [ ] テンポ: 70-90 BPM
- [ ] ドラムにスウィング/ヒューマナイズ
- [ ] サイドチェーンコンプレッション
- [ ] ハイハットのベロシティ変化

### サウンドデザイン
- [ ] ビニールノイズ/テープヒス
- [ ] ピッチワブル（テープ劣化エフェクト）
- [ ] ローパスフィルターでハイカット
- [ ] リバーブで空間の奥行き
- [ ] サチュレーション/ウォームス

### 構成
- [ ] イントロ（4-8小節）
- [ ] メインループ（8-16小節）
- [ ] バリエーション（要素の追加/削除）
- [ ] アウトロ（フェードアウトまたは4-8小節）
